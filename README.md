# 仿网易云音乐 小程序暗黑模式

## 基于taro编写
  1. 目前支持微信小程序和h5
      - h5的体验没有微信小程序好，因为taro对微信小程序支持的最好

  2. api接口来自[网易云音乐API](https://binaryify.github.io/NeteaseCloudMusicApi/#/?id=%e5%88%9d%e5%a7%8b%e5%8c%96%e6%98%b5%e7%a7%b0)
